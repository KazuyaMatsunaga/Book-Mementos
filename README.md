# Book-Mementos
書籍管理アプリケーションです．(開発途中です)

![imgur](https://i.imgur.com/2cuntLB.png)

## 使用技術

| 領域 |使用技術  |
| -------- | -------- | 
| Frontend     | Vue.js     | 
| Backend     | Go(Gin)     | 
| DB     | MySQL     | 

## 機能
- GoogleBooksAPIを用いた書籍名検索が可能
- 機能一覧
  - 書籍追加
  - 書籍削除
